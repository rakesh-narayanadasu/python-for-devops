# Python for DevOps - Quick Start Guide

## 🚀 Getting Started

### Step 1: Navigate to Course Directory
```bash
cd "c:\LEARNINGS\New folder\python-devops-course"
```

### Step 2: Start with Module 1
```bash
# Read the lesson
notepad module-01-fundamentals\01-lesson.md

# Run examples
python module-01-fundamentals\examples.py

# Try exercises
python module-01-fundamentals\exercises.py

# Check solutions
python module-01-fundamentals\solutions.py
```

### Step 3: Progress Through Modules
Follow the same pattern for each module (1-15)
